# StaySense-BI

StaySense BI is an AI-enabled Business Intelligence project developed as part of a 9-week internship program. The project focuses on transforming large-scale hotel and homestay guest reviews into meaningful business insights through data analysis, SQL, Python, visualization, and interactive dashboards.

The objective is to help hospitality businesses understand customer sentiment, identify recurring issues, monitor service quality, and support data-driven decision-making.

---

## Problem Statement

Guest reviews are often scattered across multiple platforms, making it difficult for hospitality businesses to monitor customer satisfaction and identify operational issues.

StaySense BI centralizes review data, analyzes customer feedback, and presents actionable insights through Business Intelligence techniques.

---

## Project Objectives

- Clean and preprocess large-scale hotel review data
- Perform exploratory data analysis (EDA)
- Practice SQL using real-world datasets
- Create business-focused visualizations
- Analyze guest sentiment and review trends
- Build interactive BI dashboards
- Generate business recommendations using AI-assisted analytics
