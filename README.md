## 📌 About

StaySense BI is an AI-enabled Business Intelligence project that analyzes Airbnb listing data to uncover pricing trends, customer engagement patterns, and occupancy insights.

The project combines **Python**, **Business Intelligence**, **AI-assisted analysis**, and **interactive dashboards** to help property owners make data-driven decisions.

---

## ✨ Features

- 📊 Exploratory Data Analysis (EDA)
- 🧹 Data Cleaning & Preprocessing
- 📈 Interactive Business Intelligence Dashboard
- 🤖 AI-Assisted Insight Generation
- 📖 Data Storytelling & Executive Reporting

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/jk-juhi/StaySense-BI.git
```

### 2. Navigate into the project

```bash
cd StaySense-BI
```

### 3. Create a virtual environment (optional)

```bash
python -m venv .venv
```

Activate it:

**Windows**

```bash
.venv\Scripts\activate
```

**macOS/Linux**

```bash
source .venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Launch Jupyter Notebook

```bash
jupyter notebook
```

Open the notebooks in order:

- Week01_Setup.ipynb
- Week02_EDA.ipynb
- Week03_Visualizations.ipynb
- Week04_Dashboard.ipynb
- Week05.ipynb
- Week06_AI_Insights.ipynb
